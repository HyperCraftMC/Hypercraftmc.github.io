Server Specs

Specs are very important to a good Minecraft Server. Due to donations we are thankful to allow us to continue with these amazing specs!

CPU

The current cpu we are using is a AMD Ryzen 5 3600 with 1 core.

RAM

RAM is very important to keep the server running and is a big contribution to make sure we try to keep the server running as smoothly as possible which is why we have 6GB of DDR4 Memory

Storage

It is without a dought that with a 50k world border we need a lot of storage which is why we have 50GB of NVMe SSD Storage

How do I join?

We have made is super easy to join the server! All you got to do is these simple steps.

Join the discord

Join the discord with the button below.

Discord

Find the #bot-cmds chat

Once your in the discord look for the #bot-cmds chat and type -apply, now you will see a new channel and the bot will ask you a bunch of questions. After you answer them all give the mods some time and they will look it over!

Join the server!

Once accepted you must go to #info and read all the rules in there you will also find the server IP adress and you can join from there! Have fun!
